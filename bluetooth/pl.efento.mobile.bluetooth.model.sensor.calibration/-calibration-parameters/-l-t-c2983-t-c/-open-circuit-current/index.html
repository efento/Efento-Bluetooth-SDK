enum OpenCircuitCurrent : Enum<CalibrationParameters.LTC2983TC.OpenCircuitCurrent>

Open circuit current in µA. This is the current that flows through the thermocouple when it is open-circuited. It is used to detect an open circuit condition in the thermocouple.
